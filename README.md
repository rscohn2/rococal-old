=======
rococal
=======

My calendar webapp

Create project in google cloud console: https://cloud.google.com/console
* Add google calendar api
* create key for web application
* set origin to http://127.0.0.1:9000 to use with yo

Prereq
------
install node
npm install -g yo

To clone/install
----------------
clone from git
npm install
bower install

To view
-------
grunt servebb
